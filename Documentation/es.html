Ayuda de HY-ESG-AHK

Script para usar el plugin vST HY-ESG con lectores de pantalla.

probado con:

  • HY-ESG v1.0.2
  • Reaper 6.23 (Usar con el plugin en modo bringe)
  • NVDA, Jaws for Windows, Sapi5
  • Windows10 64bits

Licencia GPL-3.0

Código fuente y últimas versiones:

https://github.com/crisspro/HY-ESG-AHK

Cómo usarlo

  1. Ejecuta HY-ESG-AHK.
  2. abre HY-ESG desde tu secuenciador.
  3. Usa los atajos de teclado del script.
  4. Puedes cerrar el script desde la bandeja del sistema

Atajos de teclado

Atajo Descripción
A Cambia al preset anterior
S Cambia al siguiente preset.
M Despliega el menú de opciones.
F1 Abre la Ayuda del script
CTRL+q Sale del script
