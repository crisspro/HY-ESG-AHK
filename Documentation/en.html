HY-ESG-AHK help

Script to use HY-ESG VST with screen readers.


    Tested with:

  • HY-ESG v1.0.2.
  • Reaper 6.23 (use bringe mode).
  • NVDA, Jaws for Windows, Sapi5
  • Windows 10 x64.

Licence GPL-3.0

Source code and last releases:

https://github.com/crisspro/HY-ESG-AHK

Usage

  1. Run HY-ESG-AHK
  2. Open HY-ESG in your DAW.
  3. Press the hotkeys of script.
  4. Can close the script from system tray

Hotkeys

Hotkey Description
M Open opnions menu.
S Next preset.
A Back preset.
F1 Open help documentation.
CTRL+q close the script.
